## Contribute an adapter

If your package implements RAIS Protocol v1, open a PR to add it to this page and the
[playground ecosystem page](https://react-ai-stream-playground.vercel.app/ecosystem).

**Checklist before submitting:**

1. Run the [Stream Inspector](https://react-ai-stream-playground.vercel.app/inspect) against your endpoint
2. Confirm all compliance checks pass (type fields, event ordering, proper termination)
3. Add a link to npm/PyPI/crates.io in your PR

```bash
# Self-test with the compliance CLI
npx rais-compliance https://your-endpoint.com/api/chat
```
